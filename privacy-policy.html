Privacy Policy

Overview

"Sell Globally from India" is an AI assistant that provides information and guidance to Indian sellers about Amazon Global Selling. This tool is built using OpenAI's GPT technology and does not collect, store, or share any user data.

Data Collection

We do not collect or store any personal information, including:

  • Login credentials or account information
  • Financial, tax, or identity-related data

Data Usage

All data entered is processed in real-time to generate relevant responses. No input is stored or reused after the session ends.

This tool may provide links to official Amazon resources (e.g., sell.amazon.in). These are for informational purposes only and we do not track link usage.

Security

No user data is collected or stored, and the assistant runs within OpenAI’s secure environment.
